## üé® Design System

### Color Gradients
```css
Primary:     from-indigo-600 to-purple-600
Success:     from-green-500 to-emerald-600
Danger:      from-red-500 to-rose-600
Warning:     from-orange-500 to-amber-600
Info:        from-blue-500 to-cyan-600
Accent:      from-cyan-500 to-blue-600
```

### Glassmorphism Recipe
```css
backdrop-blur-xl
bg-white/10 (or bg-white/5, bg-white/20)
border border-white/20
rounded-3xl (or rounded-2xl)
shadow-xl hover:shadow-2xl
```

### Animation Classes
```css
.animate-fadeIn      - Fade in with slide up
.animate-pulse       - Continuous pulse
.hover:scale-105     - Scale on hover
.hover:rotate-12     - Rotate on hover
.transition-all      - Smooth transitions
duration-300/500     - Animation timing
```

### Typography
```css
Headings:    text-5xl font-black
Subheadings: text-3xl font-bold
Body:        text-lg
Labels:      text-xs font-semibold uppercase tracking-wide
```
